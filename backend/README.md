# Setup

    Go to .env file and rename the DB_NAME to desired name

## Install dependencies

    npm i

## Run Server

    npm run dev

## Initialize database/create database

    npm run createdb

## Initialize/migrate table

    npm run latest

## Create table

    npm run make sample_table

## Delete database

    npm run deletedb

## Populate address and category table

    npm run seed
